# Praise Ziome — Portfolio

A modern personal portfolio built with HTML, CSS, and vanilla JavaScript. Features a responsive layout with a sticky navbar, mobile hamburger menu, project showcase, skills section, and contact form.

## Sections
- **Hero** — intro, stats, and call-to-action
- **Featured Project** — Smart AquaSense Dashboard
- **Projects** — selected work grid
- **Skills** — toolkit and expertise
- **About** — background and experience
- **Contact** — form and social links

## Tech Stack
HTML5 · CSS3 (custom properties, CSS Grid, Flexbox) · Vanilla JS · Font Awesome · Google Fonts (Inter)
